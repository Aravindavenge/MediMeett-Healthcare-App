# MediMeet Healthcare Appointment Booking Web Application

MediMeet is a responsive frontend healthcare appointment booking platform developed using HTML, CSS, and JavaScript. The application enables patients to register, login, search doctors by specialization, schedule appointments, and manage bookings through an interactive dashboard.

## Features
- Patient Registration and Login Authentication
- Doctor Search by Specialization
- Online Appointment Booking
- Appointment Cancellation
- Dynamic Patient Dashboard
- LocalStorage Data Management
- Responsive User Interface

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Browser LocalStorage

## Project Modules
1. Home Page
2. Patient Authentication
3. Doctor Listing
4. Appointment Booking
5. Dashboard Management
